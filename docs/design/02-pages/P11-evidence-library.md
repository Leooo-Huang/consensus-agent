# P11 — 证据库管理(v2 新增)

> 管理内外部数据源 + 查看每条证据 + 演示"V2 真接 vs P0 fixture"路线。
>
> 重要程度:**P0 必须做**(演示价值高,直接服务 L2 事实共识)
> 谁会用:业务负责人 / 路演者 / 评审
> 用什么实现:shadcn/ui Table + Card + Drawer + **in-memory JSON + cosine similarity(P0)** / **Neon pgvector(V2)**
> **变更说明(GAN-B 修)**:原文方案"sqlite-vec on Vercel Serverless"在 Vercel Functions 只读 fs + /tmp 易失环境下无法持久化。P0 fixture 数据 < 200 条,直接用 JSON + cosine 在内存计算,启动时一次性加载。V2 数据量上去后切 Neon pgvector(Neon 原生支持 pgvector extension)。

## 证据源列表

**用户怎么用**
页面左侧分组列表(图标全部用 Lucide,**禁止 emoji** — 详见 [ui.md §1.6](../../plans/2026-05-23-collab-agent-ui.md)):
- **内部数据源**(Lucide `FolderClosed` icon)
  - 历史决议(Lucide `CheckCircle2` 已接入,数量 N)
  - Demo Fixture - 七夕情侣对戒(Lucide `CheckCircle2`)
  - Demo Fixture - 百万旗舰款(Lucide `CheckCircle2`)
  - ...
  - 飞书文档(Lucide `Clock` + `--muted` 色,V2 即将支持)
  - ERP 库存数据(Lucide `Clock` + `--muted` 色,V2 即将支持)
- **外部数据源**(Lucide `Globe` icon)
  - 小红书声量(Demo Fixture)
  - 行业研报(Demo Fixture)
  - 竞品分析(Demo Fixture)
  - ...

**会发生什么**
- 点已接入的源 → 右侧加载该源下所有证据卡片
- 点未接入的源(显示 Lucide `Clock` 角标的源)→ 弹 Toast:"该数据源将在 V2 接入,黑客松阶段用 fixture 模拟"
- 路演时主动点几个 V2 源,展示路线图

**我们的规矩**
- "V2 即将支持"必须**真实显示在 UI**,不能藏起来 — 这是给评审看"我们想清楚了真生产怎么接"

**怎么算做对了**
- [ ] P0 至少 6 个 fixture 数据源可用
- [ ] V2 路线图源至少展示 4 个

## 证据卡片浏览

**用户怎么用**
右侧列表每条证据卡片显示:
- 标题
- 来源 source name + 链接
- 摘要(前 200 字)
- 标签(产品 / 市场 / 财务 等)
- "被本次推理引用 N 次"角标(若适用)
- "查看完整内容"按钮

**会发生什么**
- 点卡片 → 右侧 Drawer 显示完整证据原文 + 元数据
- 顶部"按相关度排序"开关 → 切换"按时间倒序" / "按相关度"(需要先选定一个 proposal 作为相关度基准)

**怎么算做对了**
- [ ] 每条证据有完整 metadata(来源/时间/相关度/被引次数)
- [ ] 相关度排序在选定 proposal 后正确反映

## 证据搜索

**用户怎么用**
顶部搜索框:
- 关键词搜索(Postgres LIKE)
- "用 AI 找相似"开关 → embedding 检索(经 AI Gateway 调 embedding model)

**会发生什么**
- 关键词:即时过滤
- AI 找相似:输入一段文字 → embedding → 检索 top-10 相似证据 → 显示相似度分数

**怎么算做对了**
- [ ] 关键词搜索 < 200ms
- [ ] AI 检索 < 2s 出结果

---

## 证据被引用统计

**用户怎么用**
顶部 Tab "本次推理引用了哪些证据"(从 P03/P04 跳来时显示)。展示:
- 引用次数 top 10 证据
- 哪些证据被哪些角色引用了
- 哪些证据被推理用了但**关联度 < 0.5**(可疑引用 → 标黄)

**会发生什么**
- 直接从 P03/P04 跳过来时,自动激活此 Tab
- 单击证据 → 跳回原推理位置查看上下文

**我们的规矩**
- 可疑引用必须显式标记,不能默默通过

**怎么算做对了**
- [ ] 引用统计准确
- [ ] 可疑引用有 UI 警示

## Persona 演化时间轴

**用户怎么用**
左侧 **7 个角色** Tab(运营/商品/市场/财务/品牌/供应链/**区域管理**),点任一角色 → 右侧时间轴显示该角色的:
- 创建时间 + 初始 Persona 元数据
- 历次决议回写记录(从 P09 来)
- 元数据修改记录(从 P05 来)

每个时间点显示 Before/After diff(尤其是"备注"字段的累积)。

**会发生什么**
- 加载完成 → 时间轴从下向上动画
- 点单个时间点 → 展开 diff 详情

**我们的规矩**
- 时间轴**只读** — 不允许在此页修改 Persona(去 P05 才能改)

**怎么算做对了**
- [ ] 时间轴清晰可读
- [ ] diff 高亮变化

## 1. 视觉规范

### 1.1 品牌色(基于 shadcn/ui HSL 变量)

> 调性:**企业 To B + 信任感 + 演示友好**。深色为主,关键状态用对比强烈的色提示。

| 角色 | 亮色 (HSL) | 暗色 (HSL) | CSS 变量 | 用途 |
| --- | --- | --- | --- | --- |
| 主色(品牌深蓝) | `222 47% 31%` | `217 91% 60%` | `--primary` | 按钮、链接、焦点 |
| 主色悬停 | `222 47% 26%` | `217 91% 55%` | `--primary-hover` | hover |
| 强调色(琥珀) | `38 92% 50%` | `38 92% 50%` | `--accent` | 标签、徽章、HITL 状态 |
| 支持(共识绿) | `142 71% 45%` | `142 71% 45%` | `--success` | **支持** 态度 / 共识 / Provider 正常 |
| **谨慎支持(青绿,v2.1)** | `160 60% 50%` | `160 60% 50%` | `--conditional` | **谨慎支持** 态度(v2.1 新增 4 档之一) |
| 信息不足 / 警告(琥珀) | `38 92% 50%` | `38 92% 50%` | `--warning` | **信息不足** 态度 / 降级 / 低置信 |
| 反对(冲突红) | `0 84% 60%` | `0 72% 51%` | `--destructive` | **反对** 态度 / 错误 / Provider 故障 |
| 中性灰 | `220 9% 65%` | `220 9% 46%` | `--muted` | 占位符、未激活、辅助文字 |
| 背景 | `0 0% 100%` | `222 47% 7%` | `--background` | 页面 |
| 卡片 | `0 0% 98%` | `222 33% 11%` | `--card` | 卡片、面板 |
| 边框 | `220 13% 91%` | `217 19% 24%` | `--border` | 分割线 |
| 文字-主要 | `222 47% 11%` | `213 31% 91%` | `--foreground` | 标题、正文 |
| 文字-次要 | `220 9% 46%` | `215 16% 65%` | `--muted-foreground` | 辅助 |

**v2.1 关键四档态度色语义锁定**(替代 v1 三色):

| 态度 | CSS 变量 | Lucide icon | 含义 |
| --- | --- | --- | --- |
| **支持** | `--success`(绿) | `CheckCircle2` | 强支持,无保留 |
| **谨慎支持** | `--conditional`(青绿) | `CheckCircle`(线性) | 原则同意,有前置条件 |

## 2. 动效规范(克制 + 服务可信感)

| 场景 | 动效 | 时长 | 缓动 |
| --- | --- | --- | --- |
| 页面切换 | fade + 微 translateY | 200ms | `ease-out` |
| 卡片淡入 | opacity 0→1 + Y +8px | 250ms | `ease-out` |
| 矩阵 cell 加载 | stagger 80ms 间隔 | 总 1.5s | `cubic-bezier(0.4, 0, 0.2, 1)` |
| Drawer / Modal | slide + fade | 220ms | `ease-out` |
| Toast 提示 | slide-in 右上 | 180ms | `ease-out` |
| Provider 切换 | 角标脉冲(scale 1→1.15→1) | 400ms | `ease-in-out` |
| HITL 暂停 banner | slide-down + 红色微光 | 300ms | `ease-out` |
| 流式输出 | typing-cursor 闪烁(1Hz)+ 文本渐入 | 持续 | linear |
| 按钮 hover | bg + scale 1.02 | 120ms | `ease-out` |

**严禁**:无限旋转的装饰动画 / 自动播放视频 / 多个元素同时大幅度动画。**演示场景下视觉噪音 = 失分**。

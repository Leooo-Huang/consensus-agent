## 2026-05-23 — v2 产品方向重大升级

### 触发

用户在 Phase 3 (UI/UX) 完成后,基于会议结论对产品方向做出重大升级。

### 变更摘要

**从"多 Agent 视角解读工具"升级到"基于业务方法论的企业 AI 共识形成系统"。**

### 关键变化

1. **核心机制**:引入**四层共识框架**(目标/事实/利益/权重)+ 10+ 经典方法论(OKR/RACI/Premortem/TWS(自研轨迹加权评分) 等)
2. **角色**:6 → 7(新增"区域管理"),用户也可作为 stakeholder
3. **量化**:3 档(支持/中立/反对)→ 4 档(支持/谨慎支持/反对/信息不足)
4. **数据源**:仅提案文本 → + 证据/数据引擎(内外部)
5. **决策类型识别**:无 → 自动识别(选品/营销/预算/经营)→ 动态权重
6. **反跟风算法层**:无 → TWS(自研轨迹加权评分) + Blind First-Vote + Premortem + 稳定复现测试
7. **输出**:讨论框架 → 决策报告 7 部分
8. **多端**:Web only → Web + 飞书/微信 bot (P1)

  - **Agent B**:技术可行性 + 文档间一致性 + 红线遵守
